# Contributing to SentryBridge

Thank you for your interest in SentryBridge!

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feat/your-feature`
3. Make your changes and add tests
4. Run the test suite: `npm run test:all`
5. Open a Pull Request against `main`

## Development Setup

See [README.md](README.md) for full setup instructions.

**Quick start:**

```bash
npm run install:all
npm run dev:all
```

## Code Style

- TypeScript strict mode throughout
- Test-driven development: write tests before implementation
- Follow existing module patterns (Config, PrusaLink, Camera, Obico Agent)
- Conventional Commits: `type(scope): description`

## Reporting Issues

Please use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) or
[Feature Request](.github/ISSUE_TEMPLATE/feature_request.md) templates.

## Questions

Open a Discussion on GitHub for questions and ideas.
